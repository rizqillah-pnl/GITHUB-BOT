# HACK GITHUB CONSTRIBUTiONS
## Run Locally

Clone the project

```bash
  git clone https://github.com/rizqillah-pnl/GITHUB-BOT.git
```

Go to the project directory

```bash
  cd GITHUB-BOT
```

Install dependencies

```bash
  npm install
```

Start the code

```bash
  npm run start
  #OR
  node index.js
```

    
That's it, Enjoy Code!
